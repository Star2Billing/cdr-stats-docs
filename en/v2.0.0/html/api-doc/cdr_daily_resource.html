CdrDailyResource¶

class api.cdr_daily_api.CdrDailyResource(api_name=None)¶

Attributes Details:

  • _id - contact id
  • start_uepoch - call date
  • destination_number - destination
  • hangup_cause_id -
  • switch_id - switch
